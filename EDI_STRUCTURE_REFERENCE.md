## 📄 Pricelist_36078.txt

Главният файл с всички артикули и цени.

> ⚠️ **Реалната структура е 12 колони** (EDI guide показва 11, но `SIDE` е на позиция 4 и `WEIGHT` е допълнителна последна колона).

| #   | Колона           | Описание                                      | Пример                     |
| --- | ---------------- | --------------------------------------------- | -------------------------- |
| 0   | **ITEM CODE**    | Разширен код (уникален за модел)              | `082103380`                |
| 1   | **GENUINE CODE** | Оригинален (OEM) код                          | `8-97132-835-1`            |
| 2   | **GR DESCR**     | Описание на гръцки                            | `ΠΟΡΤΑ ΚΑΡΟΤΣΑΣ`           |
| 3   | **ENG DESCR**    | Описание на английски                         | `TAIL GATE (SIDE OPENNER)` |
| 4   | **SIDE**         | Страна на частта: `LE` = ляво, `RI` = дясно   | `LE`                       |
| 5   | **MODEL CODE**   | 4-цифрен код на модела                        | `0121`                     |
| 6   | **BRAND**        | Марка на автомобила                           | `ISUZU`                    |
| 7   | **MODEL**        | Модел на автомобила                           | `ISUZU P/U 97-02`          |
| 8   | **PRICE**        | Цена за артикула (европейски формат)          | `200,00`                   |
| 9   | **CATEGORY**     | Категория на артикула                         | `TAIL BODY`                |
| 10  | **BARCODE**      | Basic item code — **JOIN ключ** към OOS/Refar | `012101480`                |
| 11  | **WEIGHT**       | Тегло на артикула                             | `1,19`                     |

> 🔑 **JOIN ключ:** `barcode` (кол. 10) = basic item code. Свързва се с `item_code` в OOS и Refar файловете.

## 📄 Outofstock_36078.txt

Наличност по складове.

> ⚠️ **Реалната структура е различна от документацията!** Вместо 2 реда на артикул (по 1 per склад), има **1 ред на артикул** с отделни колони за всеки склад.

| #   | Колона        | Описание                                   | Стойности                         |
| --- | ------------- | ------------------------------------------ | --------------------------------- |
| 0   | **ITEM CODE** | Basic item code (= barcode от Pricelist)   | `012101480`                       |
| 1   | **BARCODE**   | Баркод (= item code, обикновено идентично) | `012101480`                       |
| 2   | **ATH**       | Наличност — Атина                          | `1` = наличен, `0` = не е наличен |
| 3   | **THE**       | Наличност — Солун                          | `1` = наличен, `0` = не е наличен |

> 🔑 **JOIN:** `oos.item_code` → `pricelist.barcode`  
> ⚠️ Всички артикули се включват независимо дали са налични. Политика на компанията е да не се оповестяват реалните количества.
